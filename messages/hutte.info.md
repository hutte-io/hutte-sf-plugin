# summary

Display information about the current Hutte session.

# flags.api-token.summary

Hutte API token for authentication.

# examples

- `sf hutte info`
- `sf hutte info --api-token <token>`

# info.notLoggedIn

Not logged in. Run `sf hutte auth login` to authenticate.

# info.noDefaultProject

No default project set. Run `sf hutte project set` to configure one.
