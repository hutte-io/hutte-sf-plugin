# flags.api-token.summary

The API token. Only needed if you have not previously logged in using `sf hutte auth login`.

# flags.project-id.summary

The ID of the project. Useful when multiple projects use the same git repository.

# error.authorization

There is an error with authorization.

# error.authorization.actions

- Run `sf hutte auth login` to authenticate with Hutte.

# error.orgNotFoundOnHutte

Could not find the scratch org on Hutte. Are you sure you are in the correct project or the default org is set?

# error.noPool

This project doesn't have a pool defined.

# error.noPool.actions

- Set up a pool with at least one organization at https://app.hutte.io.

# error.noActiveOrg

There is no active pool at the moment, try again later.

# error.invalidCredentials

Invalid credentials.

# error.invalidCredentials.actions

- Verify your email and password are correct.
- Reset your password at https://app.hutte.io if needed.
